## run the project
1. Open de terminal and clone the repository
```
git clone https://github.com/Alexis-pr/PRUEBA-DE-DESEMPE-O-M-DULO-3.git
```
2. enter in the folder. 

```
cd PRUEBA-DE-DESEMPE-O-M-DULO-3.
```
3. when we are in the folder, we are used the next command:

```
code .
```

4. to install db.jSON
```
npx json-server -w db.json 
```
5. and finally used go live on index or open de file index.html in your browser
